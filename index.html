About Me

My mission is to bring cost savings and revenue generating solutions to my employers by utilizing proven data analysis techniques and technologies and to project those savings are positive cash flow into the future. For machine learning and data science projects, I use Python as my scripting language and utilize libraries such as Pandas, NumPy, Matplotlib, PyPlot, BeautifulSoup, Scikit-learn, and TensorFlow as well as the onboard libraries. For data analysis I script in SQL using PostgreSQL as my database and also am proficient in Excel and visualizations using PowerBI. I am also a web designer and build websites like this one. I am proficient in HTML, CSS, and use the Flask library in Python for back end structuring. I am a versatile all-rounder and energetic learner. If I don't know how to do something, I will learn how to do it and quickly.
